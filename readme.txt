== Installation ==

<strong>From Your WordPress Dashboard</strong>
 
1. Go to Plugins ><strong>Add New</strong>
2. Search for <strong>Modules Kit – Beaver Builder Modules</strong>
3. Click on <strong>Install Now</strong> Button
4. Click on <strong>Activate Now</strong> After Installed in the Backend 
 
<strong>From WordPress.org</strong>
 
1. Download <strong>[Modules Kit – Beaver Builder Modules](https://downloads.wordpress.org/plugin/modules-kit.zip)</strong>
2. Upload the <strong>'modules-kit’</strong> folder to the /wp-content/plugins/ directory
3. Activate <strong>Modules Kit</strong> plugin from your plugins page

== Frequently Asked Questions ==

= How do I start using Modules Kit? =

To get the full experience of the next-generation WordPress Page Builder, you’ll need the latest version of Beaver Builder installed on your website.

= Do I Need Coding Skills To Use Modules Kit? =

Not! Modules Kit for Beaver Editor is as easy to use as you can think. It works just like Beaver Builder Module, and you don’t need to code.

= Will Modules Kit Slow Down My Site? = 

Not. We built Modules Kit with performance in mind. It's built to the standards of Beaver Builder. Hence all the required styling is compiled when you use Modules.
